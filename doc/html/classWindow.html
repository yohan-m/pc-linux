Detailed Description

Main window of the GUI.

This class allows to instanciate, organize and update Qwt widgets (a 2D graphical widget and a read-only cursor) used for the display.
We can also connect or disconnect from the drone and get the drone's position in real time thanks to an udpSocket attribute.
